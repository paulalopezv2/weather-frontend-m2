Soy Paula López, estudiante del Módulo 2 de Frontend.
Este proyecto fue un desafío donde combiné:

HTML5 semántico

Bootstrap 5

JavaScript para manipular el DOM

Y diseño UI/UX usando mis propios estilos

La idea fue construir una App de Clima simulada, enfocada principalmente en la interacción, la organización del código y el diseño. Quise que mi aplicación se sintiera moderna, limpia y visualmente agradable.
Por eso decidí aplicar un diseño pastel, con sombras suaves, íconos grandes y tipografía Poppins.

Mi meta era que la experiencia fuera: Clara, minimalista, sensible al detalle, fácil de entender para el usuario y manteniendo la estructura que pide el módulo.

¿Qué hace mi aplicación?

En la página principal (Home) muestro 10 ciudades en formato de cards.

Cada card incluye un botón “Ver detalle”, que redirige según la ciudad seleccionada.

Implementé un buscador dinámico, donde las tarjetas se filtran en tiempo real.

Esta vista muestra temperatura, humedad, viento y un pequeño pronóstico semanal.

Todo se actualiza de forma dinámica según la ciudad que el usuario selecciona.
